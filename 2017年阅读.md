# 1月

1. 彼得·蒂尔 / 布莱克·马斯特斯 《从0到1-开启商业与未来的秘密》

    值得再看几遍，创业，用技术手段解决生活中不为人注意的问题，才能获得垄断地位，获得生存空间。

# 3月

2. 万维钢（同人于野） 《万万没想到：用理工科思维理解世界》

    这本书是用纸质书和iPhone上的Kindle软件同步看完的，是在用上iPhone6之后开发的新方法。这本书之前《印钞者》看到一半没有继续下去了，晚上回家要抱可可，台灯光线也暗，歇下来就是玩手机，不想拿起纸质书看，考虑用这个方法继续读完《印钞者》。

    这本书明白了自己一直不自知的错误认知。最重要一点是学习要实践，看书懂了只是肤浅的懂了，能用在生活中才是最终目的。

3. 徐瑾《印钞者：中央银行如何制造与救赎金融危机》

    这本书阅读并不流畅，很多银行金融的名词并不熟悉，读完之后对中央银行仍然概念模糊。只记住了几点：
    * 中央银行不是一开始就有的，是在经济活动中慢慢形成。
    * 中央银行应该是经济活动中最后流动性提供者。
    * 美联储是一家私人银行，日本央行、瑞士央行则是上市公司。

# 4月
4. 东野圭吾《梦幻花》（Kindle电子书）
    中国亚马逊在世界阅读日推出的活动Kindle电子书，每天2本，限时免费。亚马逊邀你2017一起读。活动起止时间：4月17日-4月30日。4月17日推出的其中一本就是东野圭吾的《梦幻花》。
    
    东野圭吾的之前看过几本，如《圣女的救济》、《新参者》等，接触更多的是由东野圭吾小说改编的电视电影等，如《神探伽利略》、《嫌疑犯X的献身》等。无论小说还是电视电影，都能轻松让你沉浸在设置的悬疑气氛里面，推理页设置的合理。
    
5. 马家辉《龙头凤尾》
    这是2016年一部热门小说，从17年1月底陆续读到4月21日方才结束，粤语用语较多，阅读时总是需要思考表达的含义，慢慢熟悉之后阅读也就流畅起来。
    
    今天早上可可又是在6点左右醒过来了，玲玲给他喂奶，我改变平时玩一会手机的习惯，拿过《龙头凤尾》继续读下去，起床吃过早饭后将书带到了办公室接着读完了。最后南爷被盟军炸弹轰的一声支离破碎结尾，确实出乎意料。之前主角光环几次大难不死后，终究逃不过乱世的命运。一段匪夷所思的乱世基情在背叛中落幕。
	
# 6月

6. 成甲《好好学习：个人知识管理精进指南》

一本谈知识管理，不仅仅是学习的书，值得看看，其中反思日记，个人觉得是最重要的一个方法。生活就是一本书，很多规律、很多道理都在生活中反复教育着我们，我们却懵懂不知，重复着同样的错误的同时只能怨天尤人。反思日记是改变这个困境的好办法。

# 9月

7. 马伯庸《古董局中局》、《长安十二时辰》

9月购买了京东的Plus会员，其中一项权益是京东阅读的畅读一年会员。大概看了下推荐的书目，选择了马伯庸的两部小说先看。这两部小说之前也有关注，一直也没有买实体书。
